Introduction

This document uses BERTopic to extract topics from open-ended survey responses about what constitutes “Good Taste” and “Bad Taste”. Since the text is short and survey-based (1-5 sentences), BERTopic’s embedding and clustering approach is highly effective. We are using the cleaned dataset (FolkTaste_CleanData.csv) which contains numeric encodings and excludes participants who were filtered out during quality checks.

Setup and data loading

We begin by loading the required packages and reading in the clean data.

import pandas as pd
from bertopic import BERTopic
from sklearn.feature_extraction.text import CountVectorizer
df = pd.read_csv("data/FolkTaste_CleanData.csv")

# The questionnaire skip pattern dictates that only people who answered YES to Taste_Possibility
# provided definitions for GoodTaste_Def and BadTaste_Def.
df_filtered = df[df['Taste_Possibility'] == 'YES'].copy()

# Extract the "GoodTaste_Def" and "BadTaste_Def" columns and drop empty responses
good_taste_docs = df_filtered["GoodTaste_Def"].dropna().astype(str).tolist()
bad_taste_docs = df_filtered["BadTaste_Def"].dropna().astype(str).tolist()

The dataset contains 224 valid responses defining “Good Taste” and 224 responses defining “Bad Taste”.
